starword-java
===================================

A native use `star` mark words for Java

# Native support

| *OS*    | *Arch*  | *JDK* |
|---------|---------|-------|
| windows | x86_64  | 8     |
| linux   | x86_64  | 8     |
| darwin  | x86_64  | 8     |
| darwin  | aarch64 | 8     |

# Usage

```xml
<project>

  <properties>
    <starword.version>0.9.8</starword.version>

    ... others
  </properties>

  <dependencies>
    <dependency>
      <groupId>org.vxwo.free</groupId>
      <artifactId>starword</artifactId>
      <version>${starword.version}</version>
    </dependency>

  ... others
  </dependencies>

  ... others
</project>
```
